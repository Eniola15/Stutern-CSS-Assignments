CSS describes how web pages are displayed and presented, it helps to style it as desired. CSS also helps with the adaptation to different devices and screen sizes.

Here i will explain the difference between paddings, margins and borders in CSS. They are all CSS Properties. A margin is the space around an element's border, while padding is the space between an element's border and the element's content. In order words the margin controls the space outside an element, and the padding property controls the space inside an element. The border is the layer of the CSS box model that sits between margin and padding. Margin and padding are always parts of an element, even if there's no visible border. The CSS box model dictates how items are displayed within your project. It defines the boundaries, spacing in and outside of the target element. In a CSS box model, the height and width are specified.This determines the size of the content area. If the padding, border and margin proprties are also specified,
then they must be added to the width and height to calculate the total size of the element.
